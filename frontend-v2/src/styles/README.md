# ADB Styles

## Shadcn

`globals.css` is generated by Shadcn and overwritten with some customizations listed below
for future reference, in case of re-initializing Shadcn.

### `body` element.

The background is customized from Shadcn defaults:

```
body {
...
-  background:
-    linear-gradient(...)
-    url('...') ...;
...
}
...
```
